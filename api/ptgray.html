PTGray based player¶

This player can play Point Gray ethernet cameras using pyflycap2.

active_settings¶

The list of settings that the camera can control.

This may only be set by calling ask_config(), not set directly.

ask_cam_option_config(setting, name, value)¶

Asks to set the setting of the camera to a specific value.

Parameters

  • setting – The setting, e.g. "brightness".

  • name – How to set it, e.g. "value" to set the value or 'one push' to auto configure it.

  • value – The value to use to set it to.

ask_config(item, ignore_play=False)¶

Asks to read the config values of the item. E.g. 'serials' for the list of serials or 'gui' to show the PTGray GUI to the user.

Parameters

  • item – The request to send.

  • ignore_play – Whether to send it event if the camera is playing.
